# UT2004 Mod Manager


## Features

* Install and manage UT2004 mods in one place
* Supports common UT2004 mod formats
* Keeps your game directory clean and organized
* Designed to reduce conflicts between mods
* Simple, fast desktop interface
* No command line usage required

---

## System Requirements

* **Windows 10 or newer**
* **Unreal Tournament 2004** installed
* No Python or extra software required (everything is bundled)

---

## Installation

1. Download the installer:

   ```
   UT2004ModManagerSetup.exe
   ```

2. Run the installer and follow the setup steps.

3. After installation, launch **UT2004 Mod Manager** from:

   * Start Menu, or
   * Desktop shortcut (if you chose to create one)

---

## First Launch

On first launch, the mod manager may ask you to:

* Locate your **Unreal Tournament 2004 installation folder**
* Confirm or create mod directories if needed

This only needs to be done once.

---

## Using the Mod Manager

Typical usage looks like this:

1. Launch **UT2004 Mod Manager**
2. Add or scan for mods
3. Enable or disable mods as needed
4. Launch UT2004 and play

The manager handles file placement so you don’t have to manually copy files into system folders.

---

## Notes & Tips

* Always close UT2004 before installing or removing mods
* If something looks wrong, restart the mod manager first
* Mods are handled in a way that avoids overwriting core game files whenever possible

---

## Uninstalling

You can uninstall at any time via:

* **Windows Settings → Apps → Installed apps**
* or **Control Panel → Programs and Features**

Uninstalling the mod manager does **not** delete your UT2004 installation.
